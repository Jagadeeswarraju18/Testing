# Spendyx Mockup Studio

This is a standalone software utility to generate professional Play Store mockups (Phone, 7" Tablet, 10" Tablet) with one click.

## Quick Start
1.  Place your app screenshots in the `input/` folder.
2.  Double-click `run.bat`.
3.  Collect your finished mockups in the `output/` folder.

## How it works
- **Unified Logic**: One script handles all device types.
- **Auto-Sharpening**: Automatically sharpens and upscales screenshots to 1080x1920.
- **Premium Hardware**: Renders high-fidelity 3D frames for the Google Pixel 9 Pro and tablets.
- **Obsidian Dark Mode**: Uses the same professional basalt/charcoal theme we designed.

## Requirements
- Python (with the `Pillow` library, which `run.bat` will try to install for you).
